About Paladeenz.

Paladeenz is an open source utility created for the game Paladins published by Hi-Rez Studios. The project is currently managed by a small team of two. The idea of the project is to utilise the API created by Hi-Rez into the game to transform data into a viewer friendly format for players. You can view our source code for the website on Github. We also strongly encourage you to join us on our Discord where you can make suggestions, report bugs, and enjoy our community.


Features you might not know about.

Since we're in active development, it's possible that we create some endpoints in Paladeenz that don't have any buttons that lead to their usage. We'll try to keep this list updated :).

  • You can create bots to pull data from the player and match lookups. Simply send a GET request to https://paladeenz.com/live?livematch=playername for livematch lookup or https://paladeenz.com/player?recentmatches=playername for recent matches. Keep in by these request limits of {{ data.LIVEMATCH_RATE_LIMIT }} and {{ data.RECENTMATCHES_RATE_LIMIT }} respectively.
  • We are currently capturing statistics for all champions and ranks. You can view the updated daily Google sheet here.
